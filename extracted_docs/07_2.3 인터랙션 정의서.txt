[UX] 2.3 인터랙션 정의서 (Interaction Definition)
프로젝트명: 영어 퀴즈 기반 NRC 리워드 플랫폼 (가칭: NRC-Walk & Quiz)
버전: v1.0
작성일: 2026. 04. 30
작성자: 제자님 (개발 총괄)
1. 개요 (Overview)
본 문서는 앱 내 주요 화면의 전환 효과, 버튼 클릭 반응, 퀴즈 정답/오답 시의 시각적 및 촉각적 피드백을 정의한다. 이를 통해 사용자에게 리워드 획득의 즐거움을 극대화하고 유려한 사용 경험을 제공하는 것을 목적으로 한다.
2. 화면 전환 애니메이션 (Transitions)
전환 시나리오
애니메이션 타입
설명
메인 → 퀴즈 진입
Slide From Right
새로운 도전이 시작되는 느낌을 주기 위해 오른쪽에서 화면이 빠르게 밀려 들어옴.
퀴즈 → 결과 화면
Fade In & Scale Up
결과가 강조되도록 서서히 나타나며 살짝 커지는 효과.
상점 상품 상세
Shared Element Transition
리스트의 상품 이미지가 상세 화면의 상단으로 부드럽게 이동하며 확대됨.
3. 리워드 피드백 (Reward Feedback) - 핵심 경험
사용자가 포인트를 얻었을 때의 쾌감을 극대화하기 위한 설정입니다.
3.1 퀴즈 정답 시 (Correct Answer)
시각 효과: 화면 중앙에 화려한 폭죽 애니메이션(Lottie 활용) 노출 + 포인트 숫자가 카운트업(0 -> 10)되는 효과.
청각 효과: 경쾌한 "딩동댕" 사운드.
촉각 효과(Haptic): 짧고 강한 진동(Success Vibration) 2회 제공.
3.2 포인트 적립 완료 (Point Accumulation)
애니메이션: 화면 하단에서 금색 코인 아이콘이 상단의 '내 포인트' 영역으로 포물선을 그리며 날아가는 효과.
결과: 상단 포인트 숫자가 부드럽게 갱신됨.
4. 퀴즈 컴포넌트 인터랙션
보기 버튼 클릭: 클릭 시 버튼이 살짝 눌리는(Scale Down to 0.95) 시각적 피드백 제공.
오답 선택 시: 버튼이 좌우로 흔들리는(Shake) 애니메이션과 함께 붉은색 테두리 노출 + 낮은 톤의 오답음.
타이머 임박: 남은 시간이 5초 미만일 때 타이머 숫자가 붉은색으로 깜빡이며 긴장감 조성.
5. 로딩 및 상태 안내 (States)
데이터 로딩: 퀴즈 데이터를 서버(FastAPI)에서 가져올 때, 정적인 로딩바 대신 '단어장이 넘어가는 애니메이션' 노출.
네트워크 오류: "서버와 연결이 원활하지 않습니다" 문구와 함께 '새로고침' 버튼이 통통 튀는 애니메이션 적용.
광고 로딩 중: "보상을 준비하고 있습니다..."라는 문구와 함께 부드러운 프로그레스 바 노출.
6. 잠금화면 특화 인터랙션
슬라이드 핸들: 사용자가 슬라이더를 밀 때 밀리는 거리에 따라 배경의 투명도가 변하여 잠금이 해제되는 느낌을 직관적으로 전달.
힌트 노출: 잠금화면 진입 시 영어 단어가 위에서 아래로 부드럽게 내려오며 시선을 끌어당김.
디자인 가이드: 모든 애니메이션은 300ms(0.3초) 내외로 설정하여 사용자가 답답함을 느끼지 않도록 '빠르고 경쾌한' 느낌을 유지함.

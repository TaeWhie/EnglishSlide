[Integration] 4.2 NRC 시스템 연동 계획서
프로젝트명: 영어 퀴즈 기반 NRC 리워드 플랫폼 (가칭: NRC-Walk & Quiz)
버전: v1.0
작성일: 2026. 04. 30
작성자: 제자님 (개발 총괄)
1. 개요 (Overview)
본 문서는 앱 내에서 적립된 리워드 포인트가 엔알씨(NRC) 커뮤니케이션의 실제 서비스 및 제품 매출로 전환되는 기술적 프로세스를 정의한다. 포인트의 가치를 실질적인 혜택으로 연결하여 고객 충성도를 높이고 본업의 성장을 돕는 것을 목적으로 한다.
2. 서비스별 연동 로직 (Service Logic)
2.1 앤텔레콤 (N-Telecom) 요금 충전 연동
연동 방식: 포인트 기반 선불 요금 충전권(PIN) 발급.
프로세스:
사용자가 앱에서 '앤텔레콤 1만 원 충전' 선택.
서버(FastAPI)는 포인트를 차감하고, 가용 PIN 번호 풀(Pool)에서 번호 추출.
사용자에게 PIN 번호 노출 및 '앤텔레콤 멤버십 앱'으로 이동 버튼 제공.
추후 과제: 앤텔레콤 API 권한 획득 시, 앱 내에서 번호 입력만으로 즉시 충전되는 Direct-API 연동 추진.
2.2 앤보임 (N-Voin) 교육 서비스 연동
연동 방식: 수강권 할인 쿠폰 또는 무료 체험권 발급.
프로세스:
퀴즈 성적이 우수한 사용자에게 '앤보임 1회 무료 체험권' 증정.
포인트 상점에서 수강료 결제 시 사용할 수 있는 '할인 코드' 발행.
해당 코드를 앤보임 결제 페이지에 입력 시 서버 간 검증(Validation) 수행.
2.3 NRC 쇼핑몰 제품 구매 연동
연동 방식: 제품별 할인 쿠폰 또는 포인트 결제 연동.
프로세스:
앱 내에서 특정 제품(예: 건강기능식품)의 '할인권' 구매.
생성된 고유 쿠폰 번호를 NRC 쇼핑몰 결제창의 '포인트/쿠폰' 란에 입력.
쇼핑몰 서버는 우리 앱 서버의 API(GET /v1/coupons/verify)를 호출하여 쿠폰 유효성 확인.
3. AI 기반 제품 추천 및 매칭 로직
제자님의 AI 역량을 활용하여 단순 노출이 아닌 '정밀 타겟팅'을 수행한다.
관심사 분석: 사용자가 자주 푸는 퀴즈 카테고리(예: 'Health & Beauty' 퀴즈 선호 시 관련 건강식품 추천) 분석.
추천 알고리즘: 사용자의 연령대, 성별, 활동량(만보기 데이터)을 기반으로 가장 필요한 NRC 제품 노출.
전환 추적: 특정 추천 상품을 클릭하거나 구매한 이력을 트래킹하여 AI 추천 모델 고도화.
4. 쿠폰 및 코드 관리 시스템
코드 생성: 중복 불가능한 16자리 영문/숫자 혼합 코드 생성 (예: NRCQ-ABCD-1234-EFGH).
보안 검증: 쿠폰 발급 시 Exchanges 테이블에 상태값을 Pending으로 저장하고, 실제 사용 완료 신호를 받아야 Completed로 변경.
만료 관리: 발행 후 30일 이내 미사용 시 자동 회수 및 포인트 환불(선택 정책) 로직 구현.
5. 데이터 동기화 및 매출 트래킹
로그 기록: 모든 포인트 전환 내역은 사용자의 NRC ID와 매칭하여 기록.
매출 보고: 매주 월요일, 지난주 앱 포인트를 통해 발생한 NRC 서비스별 실적 리포트 자동 생성.
피드백 루프: 실제 제품 구매 고객에게는 앱 내에서 '감사 포인트' 추가 지급하여 재구매 유도.
운영 전략: 앤텔레콤 실사용자가 앱을 사용할 경우 포인트 적립 가중치를 주어, 기존 고객의 해지를 방지하고 신규 고객의 앤텔레콤 번호 이동을 유도함.
